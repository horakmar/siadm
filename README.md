# SPORTident Admin

This is simple CLI software for administering of SPORTident (SI) stations.

## Features
### SI stations reading and writing: 
  * Mode
  * Code
  * Time
  * Protocol info

### Only reading:
  * Battery status
  * Firmware version

### Only Writing (commands):
  * Beep
  * Turn off
